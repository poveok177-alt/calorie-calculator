💳 Как подключить оплату

🇷🇺 ЮKassa (для России)

1. Зарегистрируйтесь на yookassa.ru
2. Подключите свой магазин (нужен ИП или самозанятость)
3. Получите shop_id и secret_key
4. Установите: pip install yookassa
5. Добавьте маршрут /buy-premium в app.py (инструкция в README)

🌍 Stripe (для всего мира)

1. Зарегистрируйтесь на stripe.com
2. Получите API ключи в Dashboard
3. Установите: pip install stripe
4. Создайте продукт и цену в Stripe Dashboard
5. Используйте Stripe Checkout — самый простой вариант

🚀 Быстрый старт без регистрации ИП

Для начала можно использовать Tribute (tribute.tinkoff.ru) или Boosty — там не нужен ИП, просто создай страницу подписки и вставь ссылку вместо кнопки выше. После получения оплаты — вручную меняй is_premium = True в базе через скрипт.
